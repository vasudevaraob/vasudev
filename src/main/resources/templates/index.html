Spring Boot Quick Start

A very famous modern and lightweight architecture based on spring suites, which helps you and your team to build web applications or web services quickly and easily.

From this project, you can learn the knowledges of Spring Framework , Spring Boot , Spring Security , Spring Data JPA , Spring HATEOAS

This project introduce the concepts of TDD, RESTful Web API, OIDC, OAuth2

System Access Control

Anonymous can access home page, login page.

User can access all page but can't access system maintain APIs

Admin can acces all pages

Custom API Access Control

Anonymous can not access any APIs

User can only retrieve(GET) service source from custom APIs

Admin can GET POST PUT DELETE service source from/to custom APIs
